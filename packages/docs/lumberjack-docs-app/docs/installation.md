Lumberjack is published as the `@lumberjackjs/angular` package.

| Toolchain | Command                             |
| --------- | ----------------------------------- |
| NPM CLI   | `npm install @lumberjackjs/angular` |
| PNPM CLI  | `pnpm add @lumberjackjs/angular`    |
| Yarn CLI  | `yarn add @lumberjackjs/angular`    |
